Change request

This portal allows you to request changes to the HR records of your employees.

Some changes can be made in Manage Self-Service (MSS) (opens in new tab). You should attempt to make these changes in MSS before submitting a request via the portal.

Changes that can be made via the portal:

  • all allowances
  • grade
  • change to pay step point
  • change to personal salary
  • unpaid leave
  • career break
  • promotions
  • term time working
  • flexible working
  • maternity, paternity, shared parental or adoption leave details
  • Extension of probation
  • removal of absence (sickness or annual leave etc)
  • annual leave amendments
  • career progression

HRSS will retain responsibility for contractors and flexi-retirement.
