Changes that can be made in Manage Self Service (MSS)

Information:

If any change is part of an existing Trac recruitment, you must complete the recruitment activity in Trac. Do not submit the changes via Manager Self Service (MSS) or the Employee Change Form (ECF).

All Managers can now process a wide range of assignment actions for their direct reports as listed below:

  • termination of employment
  • change of supervisor
  • change of hours
  • job title
  • acting up
  • extension or return from acting up
  • extension or return from secondment
  • department relocation
  • extension of fixed term contract
  • change of post
  • fixed term contract made permanent
  • changes to a secondment

All change requests received for any of the above changes will be rejected, other than changes which cannot be completed via MSS.

For example a ‘future dated change’. This will need to be detailed in the additional information box when completing the form to prevent it from being rejected.
